# Add tsk log command

Show recent ticket activity parsed from git history:
- Created tickets
- Status changes
- Comments added

Format:
```
2026-03-20  009  done      Accept flexible ticket number input
2026-03-20  009  created   Accept flexible ticket number input
```

Default to last 7 days. `tsk log <N>` shows last N days.
Show uncommitted ticket changes at the top if any.
