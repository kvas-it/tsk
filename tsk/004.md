# Build the tsk Rust CLI

Set up the Rust project and implement the initial commands:
- `tsk new` — find next ticket number, create file with front matter
  template, open `$EDITOR`
- `tsk list` — show tickets with number, title, and status
- `tsk status NNN <status>` — update a ticket's status

The CLI should find `tsk.yaml` by walking up from the current directory
and respect its configuration (dir, digits, statuses).

Keep it simple — minimal dependencies, straightforward code.
