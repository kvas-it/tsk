# Create Claude Code skill for tsk

Write a Claude Code skill that teaches Claude the tsk format so it can
create, read, and modify tickets directly.

The skill should cover:
- How to find and read tsk.yaml
- Ticket format (front matter + H1 title + body)
- How to determine the next ticket number
- Comment format and directory structure
- The "preserve unknown fields" rule

The skill should check for the `tsk` CLI and use it when available
(better listing, filtering, status changes on big projects) but fall
back to reading/writing files directly when it's not installed.
