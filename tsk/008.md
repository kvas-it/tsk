# Support negative and multi-status filters in tsk list

Allow filtering by multiple statuses and negation:
- `tsk list open in-progress` — show tickets matching either status
- `tsk list -done` — show everything except done

More reliable than `grep -v` because it matches the status field, not
words that might appear in titles.
